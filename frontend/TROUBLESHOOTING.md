# Решение проблем

## Белая страница на http://127.0.0.1:3000/

### 1. Проверьте консоль браузера
Откройте DevTools (F12) и проверьте вкладку Console на наличие ошибок.

### 2. Проверьте, что все зависимости установлены
```bash
cd frontend
npm install
```

### 3. Проверьте, что dev-сервер запущен
```bash
npm run dev
```

Должно быть сообщение:
```
VITE v5.x.x  ready in xxx ms

➜  Local:   http://localhost:3000/
```

### 4. Проверьте, что FastAPI запущен на порту 8000
```bash
uvicorn app.main:app --reload
```

### 5. Очистите кэш и переустановите зависимости
```bash
cd frontend
rm -rf node_modules package-lock.json
npm install
npm run dev
```

## Сообщение на http://127.0.0.1:8000/

Это нормально! В режиме разработки:
- Фронтенд работает на порту 3000 (Vite dev server)
- API работает на порту 8000 (FastAPI)

После сборки (`npm run build`) фронтенд будет доступен на порту 8000 вместе с API.

## Частые ошибки

### "Cannot find module"
- Убедитесь, что все зависимости установлены: `npm install`
- Проверьте, что файлы существуют по указанным путям

### "Failed to fetch" при запросах к API
- Убедитесь, что FastAPI запущен на порту 8000
- Проверьте настройки прокси в `vite.config.js`

### Белая страница без ошибок в консоли
- Проверьте, что React правильно монтируется
- Откройте Network tab в DevTools и проверьте, что все файлы загружаются
