# LegalTime Frontend

Фронтенд для системы учёта времени для юристов, выполненный в стиле Jira/JiraTempo.

## Быстрый старт

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build
```

После сборки файлы будут в папке `../static/`, откуда FastAPI будет их раздавать.

## Структура проекта

- `src/pages/` - Страницы приложения
- `src/components/` - Переиспользуемые компоненты
- `src/contexts/` - React контексты (Auth)
- `src/services/` - API клиент

## Особенности

- Монолитная архитектура - фронтенд раздаётся из FastAPI
- Стиль Jira - цветовая схема и UI элементы в стиле Atlassian Jira
- Адаптивный дизайн
- Авторизация через JWT токены
