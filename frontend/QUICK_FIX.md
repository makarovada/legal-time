# Быстрое решение проблемы белой страницы

## Шаг 1: Проверьте консоль браузера

1. Откройте http://127.0.0.1:3000/
2. Нажмите F12 (открыть DevTools)
3. Перейдите на вкладку **Console**
4. Посмотрите, есть ли ошибки (красным цветом)

## Шаг 2: Проверьте, что dev-сервер запущен

В терминале должно быть:
```
VITE v5.x.x  ready in xxx ms

➜  Local:   http://localhost:3000/
➜  Network: use --host to expose
```

Если нет - запустите:
```bash
cd frontend
npm run dev
```

## Шаг 3: Проверьте Network tab

1. В DevTools перейдите на вкладку **Network**
2. Обновите страницу (F5)
3. Проверьте, что файлы загружаются (не должно быть красных ошибок)

## Шаг 4: Очистите кэш браузера

1. Нажмите Ctrl+Shift+Delete
2. Очистите кэш и cookies
3. Или откройте в режиме инкогнито (Ctrl+Shift+N)

## Шаг 5: Переустановите зависимости

```bash
cd frontend
rm -rf node_modules
npm install
npm run dev
```

## Частые ошибки и решения

### "Failed to resolve import"
- Убедитесь, что все файлы существуют
- Проверьте правильность путей в импортах

### "Cannot read property of undefined"
- Проверьте, что все зависимости установлены
- Перезапустите dev-сервер

### Белая страница без ошибок
- Проверьте, что React правильно монтируется
- Откройте Elements tab и проверьте, есть ли элемент с id="root"
