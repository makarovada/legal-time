# Настройка фронтенда LegalTime

## Установка и запуск

### 1. Установка зависимостей

```bash
cd frontend
npm install
```

### 2. Разработка (с hot-reload)

```bash
npm run dev
```

Фронтенд будет доступен на http://localhost:3000
API запросы автоматически проксируются на http://localhost:8000/api

### 3. Сборка для продакшена

```bash
npm run build
```

Это создаст папку `static/` в корне проекта с собранными файлами.

### 4. Запуск монолитного приложения

После сборки фронтенда, запустите FastAPI:

```bash
uvicorn app.main:app --reload
```

Теперь фронтенд будет доступен на http://localhost:8000 вместе с API.

## Структура

- **Монолитная архитектура**: Фронтенд раздаётся из FastAPI как статические файлы
- **API префикс**: Все API запросы идут на `/api/*`
- **SPA роутинг**: React Router обрабатывает клиентскую маршрутизацию

## Особенности UI

- Стиль Jira: цветовая схема (#0052CC, #F4F5F7, #DFE1E6)
- Адаптивный дизайн с Tailwind CSS
- Таблицы в стиле Jira с hover эффектами
- Модальные окна для создания/редактирования
- Статусные бейджи (одобрено/черновик)

## Страницы

1. **Login** (`/login`) - Авторизация
2. **Dashboard** (`/dashboard`) - Общая статистика
3. **Time Entries** (`/time-entries`) - Учёт времени (главная страница)
4. **Matters** (`/matters`) - Управление делами
5. **Clients** (`/clients`) - Управление клиентами
6. **Contracts** (`/contracts`) - Управление договорами

## API интеграция

Все запросы идут через `src/services/api.js` с автоматическим добавлением JWT токена из localStorage.
